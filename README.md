🛒 E-commerce Dataset Analysis
📌 Overview

This project analyzes an e-commerce dataset containing 5,000 orders from customers across different cities in Egypt.

The goal is to explore sales trends, customer behavior, and key business metrics using Python.

📂 Dataset

The dataset includes:

Order ID

Customer ID

Gender

Age

City

Product Category

Product Price

Quantity

Payment Method

Order Date

Total Amount

🧹 Data Processing

Converted order_date to datetime

Extracted month and year

Created age groups

Calculated customer lifetime value (CLV)

Grouped and aggregated sales data

📊 Key Insights

989 unique customers

Debit Card is the most used payment method

July has the highest total sales

Sales by gender are almost equal

Top cities by orders: Mansoura, Aswan, Giza, Cairo, Alexandria

📈 Analysis Performed

Monthly sales trend

Sales by product category

Sales by city

Customer segmentation (Low / Medium / High spending)

Python

Pandas

NumPy

Matplotlib
